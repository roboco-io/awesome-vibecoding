# Awesome Vibe Coding (한국어)

*Language: [English](README.md) | [한국어](README.ko.md) | [日本語](README.ja.md)*

![Vibe Coding Meme](images/vibecoding-meme.png)

[Vibe Coding](https://en.wikipedia.org/wiki/Vibe_coding)은 사용자가 단순히 문제를 설명하면 AI가 필요한 코드를 생성하는 AI 기반 프로그래밍 접근 방식입니다. 개발자는 상세한 코드 로직을 깊이 이해하거나 관여할 필요가 없습니다. 프로그래머는 코딩 작업을 AI의 직관("vibes")과 능력에 맡기며, 종종 상세한 구현을 무시합니다. 이러한 마법 같은 경험은 경험이 부족한 프로그래머도 이전에는 대규모 개발팀이 필요했던 소프트웨어 솔루션을 구축할 수 있게 하며, 소프트웨어 엔지니어링 복잡성에 대한 전통적인 가정에 도전합니다.

**"Vibe Coding"** 이라는 용어는 AI 연구자 Andrej Karpathy가 2025년 2월에 만들었습니다. Karpathy는 *SuperWhisper*와 같은 고급 음성 인식 기술을 사용하여 AI와 수동적으로 상호작용하며, AI의 직관적인 코딩 능력에 전적으로 의존하는 방식을 설명했습니다. 그는 주로 지시하고 결과를 관찰하며 필요할 때만 개입하는 핸즈오프 접근 방식을 강조했습니다.

## Projects/Platforms/Tools/Services

- [Github Copilot](https://github.com/features/copilot) - GitHub Copilot은 개발자를 지원하기 위해 설계된 AI 기반 페어 프로그래머로, 실시간으로 코드와 전체 함수를 자동으로 제안합니다. GitHub와 OpenAI가 공동으로 개발한 이 도구는 고급 머신러닝을 활용하여 코드 에디터의 컨텍스트를 해석하고 지능적이고 컨텍스트 인식 제안을 제공합니다.

- [Windsurf](https://codeium.com/windsurf) - Codeium 산하에서 개발된 Windsurf는 코드 생성 및 개발자 워크플로우를 최적화하도록 설계된 AI 기반 솔루션입니다. 고급 언어 모델을 활용하여 프로그래머가 코드를 작성하고 개선하는 것을 지원하며, 프로토타이핑과 프로덕션 개발 모두를 가속화하는 원활한 경험을 제공합니다.

- [Cursor](https://www.cursor.com/) - Cursor는 소프트웨어 개발을 가속화하고 단순화하도록 설계된 AI 통합 코드 에디터입니다. 직관적인 편집 기능과 고급 생성형 AI를 결합하여, Cursor는 신입 및 숙련된 개발자 모두가 코드를 더 효율적으로 작성, 디버그 및 개선할 수 있도록 지원합니다.

- [Claude Code](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview) - Claude Code는 자연어 상호작용을 통해 코드 관련 작업을 지원하도록 설계된 Anthropic의 Claude의 특수 변형입니다. Claude의 고급 언어 기능과 개발자 중심 기능을 결합하여, Claude Code는 생산성과 소프트웨어 품질을 모두 향상시키기 위해 코드를 생성, 설명 및 개선할 수 있습니다.

- [Trae AI](https://www.futuretools.io/tools/trae-ai) - Trae AI는 개인과 기업이 고품질 콘텐츠를 만들고 브랜드 스토리텔링을 최적화하도록 돕기 위해 설계된 고급 AI 기반 플랫폼입니다. 최첨단 언어 모델을 활용하여, Trae AI는 마케팅 자료, 소셜 미디어 캠페인 또는 전략적 브랜드 커뮤니케이션을 위한 텍스트 생성 등 창작 프로세스를 간소화합니다. 직관적인 인터페이스와 데이터 기반 인사이트를 통해, Trae AI는 콘텐츠를 구상, 작성 및 개선하는 더 빠르고 스마트한 방법을 제공하여 사용자가 일관된 브랜드 목소리를 유지하고 타겟 오디언스와 공감할 수 있도록 합니다. 이 플랫폼은 또한 성과 분석을 통합하여 사용자가 참여도를 측정하고 실시간으로 전략을 최적화할 수 있도록 돕습니다.

- [AlphaCode](https://alphacode.deepmind.com/) - DeepMind가 개발한 AlphaCode는 AI 기반 코딩 분야에서 중요한 이정표를 나타냅니다. 경쟁 프로그래밍 문제를 해결하도록 설계된 AlphaCode는 고급 언어 모델을 활용하여 문제 진술을 해석하고, 고품질 코드 솔루션을 생성하며, 인간 프로그래머와 유사한 추론 단계를 수행합니다.

- [Replit Ghostwriter](https://replit.com/learn/intro-to-ghostwriter) - Replit Ghostwriter는 Replit 플랫폼에 통합된 AI 기반 코딩 어시스턴트로, 모든 기술 수준의 개발자가 코드를 작성, 디버그 및 최적화하도록 돕도록 설계되었습니다. 컨텍스트 이해와 대화형 프롬프트를 결합하여, Ghostwriter는 소프트웨어 개발 프로세스를 간소화하고 빠른 프로토타이핑을 가능하게 합니다.

- [OpenManus](https://github.com/mannaandpoem/OpenManus) - OpenManus는 학술 논문 작성 프로세스를 단순화하고 간소화하는 것을 목표로 하는 오픈소스 프레임워크입니다. 현대적인 협업 기술과 맞춤형 워크플로우를 통합하여, 연구자, 작가 및 팀이 잘 구조화된 문서를 더 효율적으로 생성할 수 있도록 지원합니다.

- [Cline](https://github.com/cline/cline) - Cline은 현대적인 개발 방식과 확장성에 중점을 둔 대화형 명령줄 환경을 제공하는 오픈소스 프로젝트입니다. 텍스트 기반 인터페이스의 전통적인 강점과 추가 도구를 결합하여, Cline은 개발자, 시스템 관리자 및 취미 개발자의 생산성을 향상시키고 간소화된 워크플로우를 제공하는 것을 목표로 합니다.

- [Duet AI for Google Workspace](https://workspace.google.com/blog/product-announcements/duet-ai?hl=en) - Duet AI는 Google Workspace 내에서 생산성을 향상시키고 워크플로우를 간소화하도록 설계된 Google의 협업형 AI 기반 솔루션입니다. 생성형 AI를 활용하여, Duet AI는 사용자가 다양한 작업을 처리하도록 돕는 것을 목표로 합니다.

- [Roo Code](https://github.com/RooVetGit/Roo-Code) - Roo Code는 에디터에 상주하는 AI 기반 어시스턴트로, 전체 개발 프로세스를 간소화합니다. 자연어 이해 기능을 갖춘 이 도구는 워크스페이스에서 직접 파일을 읽고 쓰고, 터미널 명령을 실행하며, 심지어 브라우저 작업을 자동화할 수 있습니다. OpenAI 호환 또는 맞춤형 AI 모델과 통합하여, Roo Code는 유연한 코딩 파트너나 시스템 아키텍트부터 QA 엔지니어 또는 제품 매니저와 같은 전문 역할까지 다양한 역할을 지원하는 매우 적응력 있는 코딩 환경을 제공합니다.

- [Goose](https://block.github.io/goose/) - Goose는 Block이 개발한 오픈소스 코딩 어시스턴트로, OpenAI 호환 플랫폼, Anthropic, Ollama 등을 포함한 다양한 AI 제공업체와의 원활한 통합을 통해 개발자를 지원하도록 설계되었습니다. Goose는 명령줄 또는 대화형 모드로 작동하며, Model Context Protocol (MCP)을 통해 VSCode 또는 다른 IDE와 선택적으로 통합되어 코드 생성 및 편집을 위한 강력한 환경을 제공합니다. 효율적인 워크플로우와 개발 작업에서의 다양한 지원을 제공하여 생산성을 향상시키는 것을 목표로 합니다.

- [Berrry](https://berrry.app) - Berrry는 순수 AI 직관을 통해 소셜 미디어 게시물을 기능적인 웹 애플리케이션으로 변환함으로써 vibe coding의 본질을 구현합니다. 사용자는 단순히 Twitter 게시물이나 Reddit 콘텐츠를 공유하면, Berrry의 AI가 기술적 지식이나 코드 이해 없이도 완전하고 작동하는 앱을 생성합니다. 이는 궁극적인 "vibes-only" 개발 경험입니다.

- [Vibe Compiler (vibec)](https://github.com/Strawberry-Computer/vibe-compiler) - Vibe Compiler는 프롬프트 스택을 LLM 생성을 사용하여 코드와 테스트로 변환하는 자체 컴파일 도구입니다. 마크다운 기반 프롬프트 스택을 처리하여 코드, 테스트 및 문서를 생성하며, 정적 및 동적 플러그인을 모두 지원합니다. 이 도구는 OpenRouter를 통해 Claude, ChatGPT, Grok을 포함한 모든 LLM과 작동하며, 부트스트랩 프로세스를 통해 자체적으로 컴파일할 수 있어 번호가 매겨진 단계를 통해 자체 구현을 진화시킵니다.

## Guides/Manual/Best Practices/Tutorials/Blogs

- [Vibe Coding Manual](https://www.reddit.com/r/ChatGPTCoding/comments/1j5l4xw/vibe_coding_manual/) - r/ChatGPTCoding 서브레딧의 이 Reddit 게시물은 "Vibe Coding Manual"을 탐구하며, ChatGPT로 구동되는 새로운 코딩 접근 방식에 대한 인사이트를 제공합니다. 인간-AI 협업이 어떻게 더 직관적이고 유연한 프로그래밍 방법으로 이어질 수 있는지에 대한 팁, 예제 및 개인적 경험을 공유하며, 개발자가 기술을 향상시키고 창의적인 솔루션을 발견하도록 돕습니다.

- [Here's how I use LLMs to help me write code](https://simonwillison.net/2025/Mar/11/using-llms-for-code/) - 블로그 게시물에서 Simon Willison은 대규모 언어 모델(LLM)이 코드 작성, 검토 및 설명과 같은 코딩 작업을 어떻게 도울 수 있는지 탐구합니다. 그는 잘 작동하는 프롬프트의 예를 강조하고, 부정확하거나 불완전한 출력과 같은 한계와 위험에 대해 논의하며, LLM을 개발자 워크플로우에 통합하는 효과적인 방법에 대한 인사이트를 제공합니다. Willison은 또한 정확성을 보장하고 품질을 유지하기 위해 인간의 감독이 중요함을 강조하며, AI 생성 코드에 의존하는 것의 잠재력과 도전 과제를 모두 보여줍니다.

- [Model Context Protocol (MCP): The Key To Agentic AI](https://www.youtube.com/watch?v=VChRPFUzJGA) - 이 비디오는 Model Context Protocol (MCP)에 대한 개요를 제공하며, 무엇인지, 왜 중요한지, 어떻게 작동하는지, 그리고 다른 프로토콜과 어떻게 비교되는지 설명합니다. MCP는 AI 모델, 특히 대규모 언어 모델(LLM)이 작업을 완료하는 데 필요한 데이터베이스나 파일과 같은 외부 컨텍스트에 액세스할 수 있도록 합니다. 발표자는 SQLite 데이터베이스 서버와 Claude 데스크톱을 사용한 실용적인 예제도 시연합니다.

I'll break it down for you in under 10 minutes](https://www.youtube.com/watch?v=EswVjHZMn74) - 이 콘텐츠는 AI 애플리케이션이 외부 시스템에 연결하는 표준화된 방법인 Model Context Protocol (MCP)을 소개합니다. MCP가 무엇인지, 어떻게 작동하는지, Python SDK를 사용하여 자체 MCP 서버를 구축하고 CLA 및 csor와 같은 앱에 연결하는 방법을 설명합니다. 발표자는 MCP가 개발자가 배워야 할 가치 있는 기술이라고 믿습니다.
